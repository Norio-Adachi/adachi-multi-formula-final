安達マルチ公式による円周率計算プロジェクト概要本プロジェクトは、円周率（$\pi$）の計算における演算効率の向上および、16進数表記における特定桁の抽出手法（ジャンプ計算）の探求を目的とした技術研究である。安達則男による独自の代数的アプローチに基づき、二分法を用いた計算最適化および、独自の4項公式を用いた数値的検証を行っている。構成プログラム (adachi_multi_formula_final.py)本ファイルには、本研究で得られた主要なアルゴリズムを統合している。二分法最適化ルーチン (adachi_bsplit4)従来の手法と比較し、計算速度の向上（約3.2倍）を実現した最適化アルゴリズム。16進数BBP桁抽出ルーチン (BBP_digit_extraction)独自発見の16進4項公式を用い、円周率の任意の桁へ直接アクセスする計算手法を実装。研究の背景とスタンス本研究は、既存のチュドノスキー・アルゴリズム等の数学的な最高効率手法を凌駕することを目的とするものではない。あくまで独自のアプローチを通じて、代数的操作が数値計算の振る舞いにどのような影響を与えるかを「誠実な記録」として蓄積することを主眼としている。また、無理数同士の演算における数値的循環や相関については、高精度計算を用いた検証を繰り返しているが、現時点では統計的な偶然の範疇を超えた事象は確認されていない。謝辞および注意事項本プロジェクトは、計算機科学的な検証に基づく個人の研究記録である。過度な誇張を避け、計算結果そのものの正確性を重視している。
